
# [사용자 시나리오]
	1. 사용자 합치는 이미지 1개 이상 선택
	2. 합쳐지 이미지 저장 경로 설정
	3. 시작 버튼을 통해 이미지 합친다
	4. 닫기 버튼을 통해 프로그램 종료

#[기능 명세]
	1. 파일추가 : 리스트 박스에 파일추가
	2. 선택삭제 : 리스트 박스에서 서택된 항목 삭제
	3. 찾아보기 : 저장 폴더를 서택하여 테그스 위젝에 입력
	4. 가로넓이 : 이미지 넓이 지정 (원본유지, 1024,800,600,480)
	5. 간견	    : 이미지 간의 간격 지정(없음,좁게,보통,넓게)
	6. 저장포멧 : PNG,JPG,BMP
	7  시작     : 이미지 합치기 실행
	8  진행사항 : 현재 진행중인 파일 순서에 맞게 반영
	9. 닫기     : 프로그램 종료
